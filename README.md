## Setup
Create .env file and add following variable in it:

```
PORT=<PORT>
```

To run this project, install it locally using npm:

```
$ cd ../Restaurant-App
$ npm install
$ NODE_ENV=<development OR production> npm run build
$ npm start

```

